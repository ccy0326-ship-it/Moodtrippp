# Moodtrip

Moodtrip：人工智慧之情緒導向地點與美食推薦系統

## 專案介紹

本系統根據使用者當下情緒提供景點、美食及心情小語推薦。

主要功能包含：

- 情緒選擇
- 子情緒分類
- 景點推薦
- AI模擬推薦
- 美食推薦
- 心情小語
- 收藏功能

## 開發技術

- HTML
- CSS
- JavaScript
- Python
- Flask
- JSON
- LocalStorage

## 執行方式

1. 下載專案
2. 執行 backend.py

```bash
python backend.py
```

3. 開啟 index.html

即可使用系統
